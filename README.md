# 🐾 PetShop Amigo Fiel - Fase 1

Sistema web desenvolvido para a disciplina de **Fundamentos de Sistemas Web** - Fase 1 do projeto acadêmico.

## 📋 Sobre o Projeto

Plataforma online para um petshop que oferece produtos e serviços para animais de estimação. O sistema foi desenvolvido utilizando **HTML puro** (sem CSS, Bootstrap ou JavaScript), conforme os requisitos da Fase 1 do projeto.

## ✨ Funcionalidades

- **Visualização de Produtos por Categoria:**
  - 🎀 Acessórios (roupas, brinquedos, camas)
  - 🦴 Rações não perecíveis
  - 🧴 Produtos de higiene e limpeza (tapetes higiênicos, fraldas)

- **Serviços:**
  - 🛁 Banho
  - ✂️ Tosa
  - 🛁✂️ Banho + Tosa
  - 🚗 Opção de tele-busca (com e sem)

- **Navegação completa** entre todas as páginas
- **Página de contato** com formulário

## 🛠️ Tecnologias Utilizadas

- HTML5
- Estrutura semântica (header, nav, main, footer)
- Tabelas HTML para organização de conteúdo
- Formulários HTML

## 📁 Estrutura do Projeto

```
fase1-petshop/
├── index.html              # Página inicial
├── acessorios.html         # Produtos: Acessórios
├── racoes.html             # Produtos: Rações
├── higiene.html            # Produtos: Higiene e Limpeza
├── servicos.html           # Serviços: Banho e Tosa
├── contato.html            # Página de contato
├── imagens/                # Imagens dos produtos e serviços
└── README.md               # Este arquivo
```

## 🚀 Como Visualizar

1. Clone o repositório
2. Abra o arquivo `index.html` no seu navegador
3. Ou acesse via GitHub Pages (se configurado)

## 📝 Requisitos Atendidos

✅ Cabeçalho com informações do petshop e menu de navegação  
✅ Visualização de produtos (foto, descrição, valor)  
✅ Visualização de serviços (descrição, valor, com/sem tele-busca)  
✅ Rodapé com informações de contato e autoria  
✅ Estrutura HTML semântica e bem organizada
